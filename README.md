# mod_repn_symmetric_group

Computations for the modular representation theory of the symmetric group.

https://math.mit.edu/~charchan/ModularRepresentationsSymmetricGroupSeminar.pdf

Compute the dim D^{\lambda}, the dimensions of the simple modules for the modular representations of the symmetric group by computing the rank of the Gram matrix $(\langle e_i,e_j\rangle)_{i,j=1,...,n}$ where $e_i$ is an element of the polytabloid basis.
